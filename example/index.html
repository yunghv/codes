# title:export、import 导入导出基本操作

首先第一步
import obj from 'demo.js'; // 导入对象
然后第二步
console.log(obj); // 输出对象 ``` export default { ... } ```
全部代码 
export default { ... }
 


import obj from 'demo.js'
console.log(obj)

默认导出

🔥使用 import xxx from 'filepath'进行导入（xxx 是导出的对名称，可随意命名）

注意

导出的类型是整个对象的方式

警告

使用 export 导出时避免 require 导入混用
