# nrm 及镜像源安装

全局安装 nrm

npm install -g nrm

添加私有仓库地址（verdaccio 为指定的仓库名）

nrm add verdaccio http://x.x.x.x:xxxx/

查看所有镜像源

nrm ls

切换到私有仓库

nrm use verdaccio

注册用户

npm adduser --registry

用户登录

npm login

注意

注册用户时需服务器开启注册通道
